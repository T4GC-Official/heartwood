# New Cadets: Start Here

Welcome to the Good Shepherd conservation stack. This tutorial walks you through
understanding the ecosystem before you touch any code.

Steps:
1. Read README.md in this repo
2. Read AGENTS.md (the short version)
3. Read .agent/rules/ecosystem-map.md
4. Run setup.sh to get a local copy of all repos
5. Read docs/architecture/system-overview.md
6. Pick a component and read its own README.md
